이 저장소는 **animtools**가 개발한 Eagle 플러그인의 공식 지원 채널입니다.

---

## 플러그인 목록

### Video Tile-kun 시리즈

| 플러그인 | 설명 |
|---|---|
| **Video Tile-kun** | 인스펙터 플러그인 — 동영상 썸네일을 타일 그리드로 표시합니다. |
| **VideoTileKun Player** | 타임라인 마커가 있는 전용 동영상 플레이어. |
| **Video Tile-kun Service** | 백그라운드 서비스 — 동영상 가져오기 시 썸네일을 자동 생성합니다. |

---

## 지원받는 방법

### 🐛 버그 신고

1. **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Bug Report**를 클릭하세요.
2. 템플릿을 작성하세요：플러그인 이름, Eagle 버전, OS, 재현 단계, 예상 동작과 실제 동작.

### 💡 기능 요청

1. **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Feature Request**를 클릭하세요.
2. 원하는 기능과 사용 사례를 설명해 주세요.

### ❓ 일반 문의

**[Blank Issue](https://github.com/animtools/eagle-plugin-support/issues)**를 열고 질문 내용을 작성해 주세요.

### 💬 커뮤니티（Discord）

실험적인 기능의 사전 정보・확장판 업데이트・일반 대화는 Discord 서버에서:
→ [discord.gg/c5mWhQ7nPW](https://discord.gg/c5mWhQ7nPW)

## 동작 요건

- Eagle 4.0 이상
- 일부 플러그인은 FFmpeg 플러그인이 필요합니다 — Eagle 플러그인 설정에서 설치하세요

---

## 호환성

- ✅ **Windows** — 테스트됨
- ⚠️ **macOS** — 미확인. 문제가 발생하면 피드백 환영합니다.

---

## 참고사항

- 새 Issue를 열기 전에 **기존 Issue를 검색**해 주세요.
- Issue 제목에 **플러그인 이름**을 포함해 주세요（예: `[Video Tile-kun] ...`）.
- Issue는 주기적으로 확인합니다. 응답에 며칠이 걸릴 수 있습니다.
