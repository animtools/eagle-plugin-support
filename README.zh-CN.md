## 插件列表

### VideoTileKun 系列

| 插件 | 说明 |
|---|---|
| **VideoTileKun Inspector** | 检查器插件 — 在 Eagle 检查器中以网格形式显示视频缩略图。 |
| **VideoTileKun Player** | 带时间轴标记的专用视频播放器。 |
| **VideoTileKun Service** | 后台服务 — 导入视频时自动生成缩略图。 |

## 如何获取支持

### 🐛 报告错误

1. 点击 **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Bug Report**。
2. 填写模板：插件名称、Eagle 版本、操作系统、复现步骤、预期行为与实际行为。

### 💡 功能请求

1. 点击 **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Feature Request**。
2. 描述您希望的功能及使用场景。

### ❓ 一般问题

开启 **[Blank Issue](https://github.com/animtools/eagle-plugin-support/issues)** 并描述您的问题。

### 💬 社区（Discord）

加入 Discord 服务器，获取实验性功能的早期信息、扩展版更新及一般交流:
→ [discord.gg/c5mWhQ7nPW](https://discord.gg/c5mWhQ7nPW)

## 系统要求

- Eagle 4.0 或更高版本
- 部分插件需要 FFmpeg 插件 — 请从 Eagle 插件设置中安装

---

## 兼容性

- ✅ **Windows** — 已测试
- ⚠️ **macOS** — 尚未验证。如遇问题，欢迎反馈。

---

## 注意事项

- 提交新 Issue 前，请先**搜索已有 Issue**。
- 请在 Issue 标题中注明**插件名称**（例: `[VideoTileKun Inspector] ...`）。
- Issue 会定期查看，回复可能需要数天时间。
