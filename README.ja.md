---

## プラグイン一覧

### Video Tile-kun シリーズ

| プラグイン | 説明 |
|---|---|
| **Video Tile-kun** | インスペクタープラグイン — 動画のサムネイルをタイルグリッドで表示。 |
| **VideoTileKun Player** | タイムラインマーカー付き専用ビデオプレーヤー。 |
| **Video Tile-kun Service** | バックグラウンドサービス — 動画インポート時にタイルを自動生成。 |

## サポートの受け方

### 🐛 バグ報告

1. **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Bug Report** を選択。
2. テンプレートに沿って記入してください（プラグイン名・Eagle バージョン・OS・再現手順・期待動作・実際の動作）。

### 💡 機能リクエスト

1. **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Feature Request** を選択。
2. 希望する機能とユースケースを記入してください。

### ❓ 一般的な質問

**[Blank Issue](https://github.com/animtools/eagle-plugin-support/issues)** を開き、質問内容を記入してください。

### 💬 コミュニティ（Discord）

実験的な機能の先行情報・拡張版のアップデート・雑談は Discord サーバーへ:
→ [discord.gg/c5mWhQ7nPW](https://discord.gg/c5mWhQ7nPW)

## 動作要件

- Eagle 4.0 以降
- 一部プラグインは FFmpeg プラグインが必要です（Eagle のプラグイン設定からインストール）

---

## 動作確認環境

- ✅ **Windows** — テスト済み
- ⚠️ **macOS** — 未確認。お気づきの点はフィードバック歓迎です。

## 注意事項

- 新規 Issue を開く前に、**既存の Issue を検索**してください。
- Issue タイトルに**プラグイン名**を含めてください（例: `[Video Tile-kun] ...`）。
- Issue は定期的に確認しています。返答まで数日かかる場合があります。
