---

## Плагины

### Серия VideoTileKun

| Плагин | Описание |
|---|---|
| **VideoTileKun Inspector** | Плагин инспектора — отображает миниатюры видео в виде сетки плиток. |
| **VideoTileKun Player** | Специализированный видеоплеер с маркерами шкалы времени. |
| **VideoTileKun Service** | Фоновая служба — автоматически создаёт плитки при импорте видео. |

## Как получить поддержку

### 🐛 Сообщение об ошибке

1. Нажмите **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Bug Report**.
2. Заполните шаблон: название плагина, версия Eagle, ОС, шаги для воспроизведения, ожидаемое и фактическое поведение.

### 💡 Запрос функции

1. Нажмите **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Feature Request**.
2. Опишите функцию и ваш сценарий использования.

### ❓ Общие вопросы

Откройте **[пустой Issue](https://github.com/animtools/eagle-plugin-support/issues)** и опишите свой вопрос.

### 💬 Сообщество (Discord)

Присоединяйся к серверу Discord для раннего доступа к экспериментальным функциям, обновлениям расширенной версии и общения:
→ [discord.gg/c5mWhQ7nPW](https://discord.gg/c5mWhQ7nPW)

## Требования

- Eagle 4.0 или выше
- Некоторые плагины требуют плагин FFmpeg — установите его в настройках плагинов Eagle

---

## Совместимость

- ✅ **Windows** — протестировано
- ⚠️ **macOS** — ещё не проверено. Отзывы приветствуются при обнаружении проблем.

## Примечания

- Пожалуйста, **ищите в существующих Issues** перед открытием нового.
