## 插件列表

### VideoTileKun 系列

| 插件 | 說明 |
|---|---|
| **VideoTileKun Inspector** | 檢查器插件 — 在 Eagle 檢查器中以網格形式顯示影片縮圖。 |
| **VideoTileKun Player** | 帶時間軸標記的專用影片播放器。 |
| **VideoTileKun Service** | 背景服務 — 匯入影片時自動產生縮圖。 |

## 如何取得支援

### 🐛 回報錯誤

1. 點選 **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Bug Report**。
2. 填寫範本：插件名稱、Eagle 版本、作業系統、重現步驟、預期行為與實際行為。

### 💡 功能請求

1. 點選 **[Issues](https://github.com/animtools/eagle-plugin-support/issues) → New Issue → Feature Request**。
2. 描述您希望的功能及使用情境。

### ❓ 一般問題

開啟 **[Blank Issue](https://github.com/animtools/eagle-plugin-support/issues)** 並描述您的問題。

### 💬 社群（Discord）

加入 Discord 伺服器，搶先獲取實驗性功能資訊、擴充版更新及一般交流:
→ [discord.gg/c5mWhQ7nPW](https://discord.gg/c5mWhQ7nPW)

## 系統需求

- Eagle 4.0 或更新版本
- 部分插件需要 FFmpeg 插件 — 請從 Eagle 插件設定中安裝

---

## 相容性

- ✅ **Windows** — 已測試
- ⚠️ **macOS** — 尚未驗證。如遇問題，歡迎回饋。

---

## 注意事項

- 提交新 Issue 前，請先**搜尋現有 Issue**。
- 請在 Issue 標題中註明**插件名稱**（例: `[VideoTileKun Inspector] ...`）。
- Issue 會定期查看，回覆可能需要數天時間。
